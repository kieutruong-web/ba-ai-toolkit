# ba-ai-toolkit
AI-assisted toolkit for Business Analysts (User Stories, Acceptance Criteria, Test Cases)

## Overview

This project demonstrates how AI can support Business Analysts in producing clear, consistent, and structured documentation.

## Structure

- `docs/` – Documentation templates  
- `prompts/` – AI prompts for generating content  
- `examples/` – Sample outputs  

## Purpose

- Improve documentation quality
- Standardize BA artifacts
- Leverage AI in daily BA work

## Author

Business Analyst with experience in technical writing and AI-assisted workflows.
